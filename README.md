
# RandomTeamSelector

The mobile application is RandomTeamSelector offers a wheel to randomly select teams in games such as Pes and Fifa.

The app is made entirely in UIKit.

**`Dependencies:`** 
<p>Firebase</p>
<p>Snapkit</p>
<p>GoogleMobileAds</p>
<p>SwiftFortuneWheel</p>
